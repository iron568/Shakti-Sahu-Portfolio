# Shakti Sahu Portfolio

This project contains both the frontend (React + Vite + Tailwind) and backend (Node.js + Express + MongoDB) for Shakti Sahu's portfolio.

## Frontend

- Built with React, Vite, and Tailwind CSS.
- Main files: `src/App.jsx`, `src/pages/`, `src/components/`, `index.html`.
- To start:
  ```bash
  cd frontend
  npm install
  npm run dev
  ```

## Backend

- Built with Node.js, Express, and MongoDB.
- Main files: `server.js`, `routes/contact.js`, `models/Contact.js`, `config/db.js`.
- To start:
  ```bash
  cd backend
  npm install
  npm run dev
  ```
- Configure your MongoDB URI in `backend/.env`.
